Pantalla de Administrador: Panel de Control

1. Gestión de Empleados
   - Lista de empleados:
     - Nombre, posición, y estado actual (presente/ausente).
   - Acciones:
     - Agregar, editar o eliminar empleados.
     - Ver detalles de un empleado (incluyendo historial de asistencia).

2. Historial de Asistencia
   - Filtros:
     - Por rango de fechas.
     - Por empleado.
     - Por tipo de registro (entrada/salida).
   - Tabla:
     - Fecha, hora, tipo (entrada/salida), empleado.
   - Exportar datos:
     - Botón para exportar el historial en formatos como CSV o PDF.

3. Gestión de Asistencia
   - Registrar asistencia manualmente:
     - Permitir al administrador agregar o corregir registros de asistencia.
   - Validar registros:
     - Revisar y aprobar registros sospechosos o incompletos.

4. Configuración
   - Parámetros del sistema:
     - Horarios laborales.
     - Tolerancia para llegadas tarde.
   - Roles y permisos:
     - Gestionar roles (administrador, empleado) y sus permisos.

Flujo de Navegación
   - Inicio: Resumen del Dashboard.
   - Empleados: Gestión de empleados.
   - Asistencia: Historial y gestión de registros.
   - Configuración: Ajustes del sistema.

Pantallas para el Empleado

1. Registro de Asistencia
   - Descripción: Permitir al empleado registrar su entrada y salida.
   - Contenido:
     - Mostrar el nombre del usuario autenticado.
     - Botones para registrar:
       - Entrada: Registrar la hora de entrada.
       - Salida: Registrar la hora de salida.
     - Validaciones:
       - Verificar que el usuario esté autenticado.
       - Mostrar mensajes de confirmación al registrar la asistencia.

2. Perfil del Empleado
   - Descripción: Mostrar información personal del empleado.
   - Contenido:
     - Información personal:
       - Nombre.
       - Correo electrónico.
       - Posición.
       - Roles actuales.
     - Historial de roles y permisos asignados.
     - Botón para actualizar información personal (si está permitido).

Flujo de Navegación
- Inicio: Resumen del perfil del empleado.
- Registro: Pantalla para registrar entrada y salida.
